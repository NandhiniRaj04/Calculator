## Getting Started

Follow these steps to run the app locally:

### Prerequisites

- Ruby 3.2.2
- Bundler (`gem install bundler`)


### Installation

```bash
git clone https://github.com/NandhiniRaj04/Calculator.git
cd calculator
bundle install

### Run the application 
 rails s
